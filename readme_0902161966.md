# Procedimiento
1. Creamos un Folk 
2. Ejecutamos el comando git clone "url del respositorio"
3. Creamos una rama con el comando git checkout -b
4. Relizamos los cambios al proyecto
5. Agregamos los archivo con git add .
6. Subimos los cambios a nuestra rama con git push 
7. Unificados las ramas con git merge
8. Procedemos a subir los archivos con git push
9. Realizamos un Pull Request
